CUTEWRITER | APRIL 2016

CuteWriter is a notepad clone based around a "cute" theme. The project is complete and fully functional.

It was written in C# and is a WPF application designed with XAML. CuteWriter was an app that I very much enjoyed creating, and something that I learned a lot from. Working with files was fun, and I'm looking forward to using these skills in the future.

This page is still under construction. Feel free to check out my source code here, compile it yourself and have a go!
