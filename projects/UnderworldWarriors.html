Underworld Warriors

Underworld Warriors is a slow paced, combat and exploration oriented text-based role playing game. It's my first attempt at delving into the world of video game development. I decided to start simple by using technologies I am already very familiar with such as Visual Studio and languages like C# and XAML. It's improving every single day, and I hope to release the final version by the end of the year.

This page is still under construction and the project is not yet complete. Feel free to check out my source code here, compile it yourself and have a go!
